Syria YouTube Monitor - About
This is a prototype application for monitoring trending videos related to the Syrian conflict. This for the interested person to see what kind of content is being uploaded on a daily basis, how this relates to current events and to see how popular each piece of content is as well as the kind of discussions happening through comments on the video. This tool does not start with an ideological perspective a priori, rather it is intended to be an informational tool to better document and understand user generated content independently from the origin.

From a technological perspective, a daily query is made to the YouTube public API asking for videos uploaded in the previous 24 hours that match a given keyword or phrase. The match may be in the video's title, description or summary. The public information on the uploading user, comments and comment authors are also taken from separate API calls.

The web interface allows interaction with the data stored in MongoDB. The pages are served using the Flask library with the PyMongo connector. Animations and charts use jQuery and D3 respectively.
